--- Title: Support heading: Support for CoEpi ---

Need help with CoEpi?

  1. First, check out the issues list for either Android or iOS, and see if your issue has already been reported.

  2. If not, feel free to create a new issue (click here to start for Android; click here to start for iOS) so we can most easily track resolution of any issues. You can also add a feature request using this method.

  3. Not able to use Github? You can also hit "Report a problem" within the app, under your settings screen, to automatically generate an email to us. Feel free to include any screenshots if it would be helpful for sharing what you see.
